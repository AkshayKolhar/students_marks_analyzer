# Pandas Sample Project

This project demonstrates a simple data analysis workflow using Python and Pandas.

## What it does

- Loads student data from a CSV file
- Calculates the average score for each course
- Identifies the top-performing student
- Saves the summary output to a new CSV file

## Files

- `pandas_project.py` - Main script containing the data loading and analysis logic
- `students.csv` - Input dataset with student records
- `student_summary.csv` - Output file generated after running the script

## Requirements

Make sure you have Python installed, along with the Pandas package:

```bash
pip install pandas
```

## How to run

Run the script from the project folder:

```bash
python pandas_project.py
```

The program will print the student data, course averages, and top performer information, and will also save the summary to `student_summary.csv`.

## Purpose

This repository is a beginner-friendly example of using Pandas for basic data analysis and CSV processing.
